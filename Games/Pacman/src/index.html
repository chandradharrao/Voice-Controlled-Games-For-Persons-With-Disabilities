How to Play?

Click on Mouse to enbale sound.

  • To use Voice Commands,Say the Letter P to Start playing and Say the below letters to Control PACMAN:
    • "UP" - W
    • "LEFT" - A
    • "BOTTOM" - S
    • "RIGHT" - D

  • To use WASD keyboard movement,PRESS P to Start!
